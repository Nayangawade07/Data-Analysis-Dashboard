Super Store Sales (Interactive Power BI dashboard with KPIs, slicers, and data-driven insights)

## Project Objective
The objective of this project is to design and develop an interactive Power BI dashboard to analyze business performance and provide meaningful insights. The dashboard helps stakeholders understand key metrics, trends, and patterns for better data-driven decision-making.

## Dataset Used
The dataset contains business-related data such as:
- Sales
- Profit
- Quantity
- Orders
- Product and category details
- Date-related fields
The data was cleaned and transformed using Power Query before creating the dashboard.
  <a href="https://github.com/Nayangawade07/Data-Analysis-PowerBi Dashboard/blob/main/Dashboard_Demo.mp4">Super Store Dashboard Demo</a>
  

## Key Questions (KPIs)
The dashboard answers the following key business questions:
- What is the total sales and total profit?
- How many orders have been placed?
- Which products or categories perform the best?
- What are the sales and profit trends over time?
- How does performance change based on slicers and filters?

Key KPIs used:
- Total Sales
- Total Profit
- Total Orders
- Quantity Sold
- Average metrics (where applicable)

## Dashboard Process
1. Imported the dataset into Power BI  
2. Cleaned and transformed data using Power Query  
3. Created relationships and data model  
4. Built DAX measures for KPIs  
5. Designed interactive visuals such as cards, charts, and slicers  
6. Optimized the dashboard layout for better readability

- Trend analysis helps identify high-performing and low-performing areas  
- Visuals make complex data easy to understand
